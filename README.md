# ananicy-cpp-rules
Ananicy daemon with community-driven rules of CachyOS
Ananicy (ANother Auto NICe daemon) is a shell daemon created to manage
processes' IO and CPU priorities.
.
This package includes the community-driven set of rules for popular 
applications from CachyOS, optimized for the AnduinOS environment. Anyone 
may add their own rule via GitHub's pull request mechanism.
